JavaScript: Tổng quan và cách thêm script vào trang web

JavaScript là ngôn ngữ lập trình chạy trên trình duyệt, giúp tương tác với HTML/CSS và tạo các hiệu ứng động. Dưới đây là hướng dẫn chi tiết về cách nhúng script vào trang web.

1. Giới thiệu JavaScript

  • Đặc điểm:

    • Chạy trực tiếp trên trình duyệt.
    • Tương tác với HTML, CSS và DOM.
    • Hỗ trợ lập trình hướng đối tượng, hàm, sự kiện.

  • Cách nhúng vào HTML:

    1. Inline Script:
    <button onclick="alert('Hello!')">Click me</button>
    1. Internal Script:
    <script>
  console.log('Hello from internal script');
</script>
    1. External Script:
    <script src="script.js"></script>
    • script.js chứa các lệnh JavaScript.

2. Cách thêm script vào trang web

  • Inline Script: viết trực tiếp trong thẻ HTML, dùng cho script ngắn.
  • Internal Script: viết trong thẻ <script> của HTML, phù hợp thử nghiệm hoặc trang nhỏ.
  • External Script: viết trong file .js riêng, dễ quản lý và tái sử dụng.
  • Ví dụ minh họa External Script:

script.js:

function greet() {
    alert('Hello from external script!');
}

HTML:

<button onclick="greet()">Click me</button>
<script src="script.js"></script>
  • Có thể dùng thuộc tính defer để script chạy sau khi HTML load xong: <script src="script.js" defer></script>

3. Kết luận

  • JavaScript là công cụ mạnh mẽ để tương tác với trang web và tạo trải nghiệm người dùng động.
  • Chọn cách nhúng script phù hợp với quy mô dự án và quản lý code hiệu quả: Inline cho nhanh, Internal cho thử nghiệm, External cho dự án lớn.
